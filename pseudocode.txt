На экран выводится вертикальный нумерованый список с категориями задач.
На экран выводится строка с текстом "Выберите из списка категорию задачи: ".
Пользователь вводит цифру категории из нумерованого списка. 
Из списка по индексу берется элемент и записывается в переменную.

На экран выводится вертикальный нумерованый список с уровнями сложности задач.
На экран выводится строка с текстом "Выберите уровени сложности: ".
Пользователь вводит цифру сложности из нумерованого списка. 
Из списка по индексу берется элемент и записывается в ту же переменную что и категория задачи.

На экран выводится вертикальный нумерованый список с ЯП.
На экран выводится строка с текстом "Выберите из списка ЯП: ".
Пользователь вводит цифру ЯП из нумерованого списка. 
Из списка по индексу берется элемент и записывается в ту же переменную что категория задачи и сложность.

Потом по API отправляется ИИ запрос с данными из переменной.
ИИ генерирует еще несколько вопросов для уточнения задачи.

Пользователю на экран выводится текст с вопросами ИИ.
Он отвечает на вопросы. Ответы записывается в переменную.

Потом по API отправляется ИИ запрос с данными из переменной.
ИИ генерирует ТЗ.
ТЗ выводится на экран.
